# Desafio-CRUD
el programa trata de un menú sobre pokemones,  consta de 5opciones para poder ejecutar el programa que funciona dentro de un While True, lo que hace que la consola siga todas las órdenes que se le den.
El menú permite ingresar(tipo, nombre, poder y nivel), leer, administrar y eliminar los pokemones, también tiene la opción de salir. En este programa se utilizaron códigos vistos en el curso de programación de cisco como .strip y none
